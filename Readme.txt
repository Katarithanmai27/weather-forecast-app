A responsive weather forecast web app built with **JavaScript**, **HTML**, **Tailwind CSS**, and the **OpenWeatherMap API**. The app allows users to:

- Search weather by city name
- Get weather for current location
- View 5-day extended forecast
- See recent city search dropdown (stored in localStorage)
- Handle invalid inputs and API errors gracefully

---

##  Features

-  City and Location-Based Weather Forecast
-  5-Day Weather Forecast with Date, Icon, Temp, Wind, and Humidity
-  Smart City Search Dropdown (localStorage powered)
-  Fully Responsive for desktop, tablet, and mobile devices
-  Error Handling and Input Validation
-  Clean UI using Tailwind CSS

---

##  Tech Stack

- HTML5
- Tailwind CSS
- JavaScript
- [OpenWeatherMap API](https://openweathermap.org/api)
- LocalStorage
